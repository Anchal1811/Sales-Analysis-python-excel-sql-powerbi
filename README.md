## 📈 Business Insights & Key Metrics

This section highlights the direct business value generated by this technical pipeline.

### 1. Improved Data Reliability
* **Insight:** Processed 520 raw records and identified 38 data errors (20 duplicates and 18 missing dates) using Python. This refined the dataset down to **482 clean rows**.
* **Business Value:** Prevents **over-reporting revenue by ~7%**. This ensures management is not making critical decisions on inventory or staffing based on inaccurate, inflated sales numbers.

### 2. Profit Margin Tracking (DAX)
* **Insight:** Established an automated calculation of row-level profit margin using a custom DAX measure in Power BI: `[Profit] / [Sales]`.
* **Business Value:** Provides instant visibility into low-margin products or regions. Management can now pinpoint if low profits are a pricing issue or an operational cost issue.

### 3. Regional Performance Analysis
* **Insight:** The interactive dashboard breaks down all performance data by the four key sales territories.
* **Business Value:** Enables efficient resource allocation—marketing budgets can be shifted to high-growth regions, and supply chain logistics can be optimized in under-performing ones.

### 4. Seasonality and Trend Detection
* **Insight:** Extracted `Year` and `Month` features in Python, allowing time-series visualization in the dashboard across multiple years.
* **Business Value:** Helps optimize inventory planning and staffing levels by identifying seasonal peaks and valleys.
